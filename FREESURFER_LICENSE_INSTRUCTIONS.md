# FreeSurfer License

To run fMRIPrep, you need a FreeSurfer license. It is free but required.

1.  **Register here:** https://surfer.nmr.mgh.harvard.edu/registration.html
2.  **Receive Email:** They will send you a short text block.
3.  **Save File:** Copy that text into a file named `license.txt` in this directory.

*Note: Do not commit your actual license file to public repositories if you want to keep it private, though it is generally free.*
